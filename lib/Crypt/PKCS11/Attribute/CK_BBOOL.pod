=encoding utf8

=head1 NAME

Crypt::PKCS11::Attribute::CK_BBOOL - A CK_BBOOL PKCS #11 attribute.

=head1 SYNPOSIS

  # Create an attribute module with a CK_BBOOL value
  package Crypt::PKCS11::Attribute::Token;
  use base qw(Crypt::PKCS11::Attribute::CK_BBOOL);
  use Crypt::PKCS11 qw(:constant);
  sub type () { CKA_TOKEN }

  # Use the attribute
  my $attribute = Crypt::PKCS11::Attribute::Token->new;
  $attribute->set(1);

=head1 DESCRIPTION

A PKCS #11 attribute containing a CK_BBOOL value.

=head1 METHODS

See also `Crypt::PKCS11::Attribute` for the base module methods.

=over 4

=item $attribute = $attribute->set ($bool)

Sets the CK_BBOOL value within the attribute. May croak on error and returns
itself on success.

=over 4

=item $bool

Must be defined and set to true (!0) or false (0).

=back

=item $bool = $attribute->get

Return the CK_BBOOL value within the attribute or undef if not set.

=back


=encoding utf8

=head1 NAME

Crypt::PKCS11::CK_RSA_PKCS_OAEP_PARAMS - Perl interface to PKCS #11 CK_RSA_PKCS_OAEP_PARAMS structure

=head1 SYNPOSIS

  use Crypt::PKCS11::CK_RSA_PKCS_OAEP_PARAMS;
  my $obj = Crypt::PKCS11::CK_RSA_PKCS_OAEP_PARAMS->new;
  $obj->set...;
  $obj->get...;

=head1 DESCRIPTION

This is the Perl interface for the C structure CK_RSA_PKCS_OAEP_PARAMS in PKCS #11.
See PKCS #11 documentation for more information about the structure and what it
is used for.

=head1 METHODS

=over 4

=item $obj = Crypt::PKCS11::CK_RSA_PKCS_OAEP_PARAMS->new

Returns a new Crypt::PKCS11::CK_RSA_PKCS_OAEP_PARAMS object.

=item $rv = $obj->get_hashAlg($hashAlg)

Retrieve the value B<hashAlg> from the structure into C<$hashAlg>.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $hashAlg = $obj->hashAlg

Returns the value B<hashAlg> from the structure or undef on error.

=item $rv = $obj->set_hashAlg($hashAlg)

Set the value B<hashAlg> in the structure.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $rv = $obj->get_mgf($mgf)

Retrieve the value B<mgf> from the structure into C<$mgf>.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $mgf = $obj->mgf

Returns the value B<mgf> from the structure or undef on error.

=item $rv = $obj->set_mgf($mgf)

Set the value B<mgf> in the structure.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $rv = $obj->get_source($source)

Retrieve the value B<source> from the structure into C<$source>.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $source = $obj->source

Returns the value B<source> from the structure or undef on error.

=item $rv = $obj->set_source($source)

Set the value B<source> in the structure.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $rv = $obj->get_pSourceData($pSourceData)

Retrieve the value B<pSourceData> from the structure into C<$pSourceData>.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=item $pSourceData = $obj->pSourceData

Returns the value B<pSourceData> from the structure or undef on error.

=item $rv = $obj->set_pSourceData($pSourceData)

Set the value B<pSourceData> in the structure.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=back

=head1 PRIVATE METHODS

These are the private methods used within the module and should not be used
elsewhere.

=over 4

=item $bytes = $obj->toBytes

Return the structure represented as bytes or undef on error.

=item $rv = $obj->fromBytes($bytes)

Sets the structure from a representation in bytes.
Returns C<CKR_OK> on success otherwise a CKR describing the error.

=back
